## Overview

The Resume Builder App helps users design resumes effortlessly. Users can select templates, fill in their personal details, preview the resume, and download it in PDF format. The app provides a structured workflow, making resume creation simple for users of all experience levels.

---

## Features

* **User-friendly Interface:** Intuitive design for all skill levels.
* **Multiple Templates:** Choose from predefined resume templates.
* **Customizable Sections:** Add/remove sections, modify fonts, and change colors.
* **Easy Editing:** Make changes to your resume at any stage.
* **Download & Print:** Export resumes as PDF or print directly.

---

## Pages

### 1. Home Page

* Displays a **navigation bar** with logo and links:

  * Resume Template
  * My Resume
  * About Us
* Shows **4 predefined resume templates**.
* Hovering over a template reveals a **"Use Template"** button to proceed to details filling.

### 2. Details Filling Page

* Split into **sidebar navigation** and **form layout**.
* Default section: **Personal Info**.
* Navigate to other sections using the sidebar or **Next** button:

  * Work Experience
  * Education
  * Key Skills

### 3. My Resume Page

* Preview your created resume.
* Buttons:

  * **Go Back:** Return to Key Skills page for edits.
  * **Save Resume:** Download resume and show success modal.

### 4. About Us Page

* Provides information about the app and its purpose.
